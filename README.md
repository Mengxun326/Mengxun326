你好，我是 **MengXun**，来自安徽理工大学信息安全专业的学生开发者。

我喜欢把课堂知识、工程实践与真实需求结合起来，从需求分析、架构设计、编码测试到发布维护，完整地把一个想法做成可用的软件。目前主要探索：

## ✨ 当前重点：Vexra

**Vexra** 希望为应用提供类似 SQLite 的本地优先使用体验，同时加入现代向量检索能力：

* 单文件持久化与 WAL 故障恢复
* Flat 精确检索与 HNSW 近似最近邻检索
* Cosine、Euclidean 与 Dot Product 距离度量
* JSON 元数据存储与过滤表达式
* BM25、向量检索与 RRF 融合组件
* AVX2 / NEON SIMD 距离计算路径
* Rust API、CLI、HTTP、Python、JavaScript 与 C ABI
* 内置本地 Dashboard，适合桌面应用、RAG 原型和边缘部署
